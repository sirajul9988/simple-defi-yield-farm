# Simple DeFi Yield Farm

A streamlined, educational Web3 project demonstrating the core mechanics of Yield Farming. Users can stake a dummy LP token (or any ERC-20) to earn rewards over time.

## Features
- **ERC-20 Reward Token:** A standard mintable token implementation.
- **Staking Vault:** Logic for depositing, withdrawing, and claiming rewards.
- **Flat Architecture:** All files are located in the root directory for easy deployment and study.
- **Web Interface:** Vanilla HTML/JS frontend using Ethers.js.
